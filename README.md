Franks Flower Shop Landing Page

A responsive single-page product landing page for a fictional flower shop created as part of a web development project. This website showcases featured flower products, shop information, and an interactive shopping cart system using HTML, CSS, and JavaScript.

Project Objective

To design and develop a responsive, single-page product landing page and manage the development lifecycle using Git/GitHub.

Features
- Responsive Navigation Bar with Smooth Scrolling
- Hero Section with Shop Introduction
- About Us / Features Section
- Product Grid Layout for Flower Bouquets
- Interactive Add to Cart Functionality
- Sliding Cart Panel
- Automatic Quantity Counter for Duplicate Products
- Dynamic Total Price Calculation
- Buy Now Checkout Alert
- Contact Section with Font Awesome Icons
- Responsive Design for Desktop and Mobile

Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Git / GitHub

Interactive Elements Implemented

1. Add to Cart Button
   - Adds selected product to cart
   - Increases quantity for duplicate items

2. Sliding Cart Panel
   - Toggleable cart sidebar for viewing selected items

3. Buy Now Button
   - Validates cart before purchase
   - Displays purchase confirmation alert
   - Clears cart after checkout

Responsive Design

This landing page uses:

- CSS Grid for product layout
- Flexbox for section alignment
- Media Queries for mobile responsiveness
